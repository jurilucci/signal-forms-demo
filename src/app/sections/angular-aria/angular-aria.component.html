Angular Aria

Cos'è

Angular Aria (@angular/aria) è un pacchetto che fornisce direttive headless e pronte per l'accessibilità, basate sui pattern WAI-ARIA.

  • Gestione tastiera, attributi ARIA, focus e supporto screen reader.
  • Nessuno stile imposto: puoi applicare il tuo design (Tailwind, Material, custom).
  • Pattern inclusi: accordion, combobox, grid, listbox, menu, tab, toolbar, tree.
