4. Schema e riuso

Come funziona

Lo schema è un insieme di regole (validatori, disabled, readonly, ecc.) definito una volta e applicato in più punti con apply(path, schema).

  • schema<T>(fn) – definisci le regole per un tipo (es. Address).
  • apply(customer.billingAddress, addressSchema) – applica lo schema al campo annidato.
  • Per array: applyEach(order.items, itemSchema) applica lo schema a ogni elemento.

Qui lo stesso addressSchema è usato per indirizzo di fatturazione e di spedizione.
