6. Custom control (FormValueControl)

Come funziona

Invece di ControlValueAccessor, con i Signal Forms basta implementare FormValueControl<T>:

  • Proprietà obbligatoria: readonly value = model<T>(...) (ModelSignal).
  • Opzionali: disabled, touched, errors, invalid, name, required, ecc. Se presenti, la direttiva [formField] le compila automaticamente.
