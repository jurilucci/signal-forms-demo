Control flow e template

Nuova sintassi built-in

Angular offre il control flow nel template con blocchi integrati: @if, @for, @switch, @defer.

  • @if (cond) { ... } / @else — condizionale senza *ngIf.
  • @for (item of items; track item.id) — loop con track obbligatorio, migliore performance.
  • @switch (value) con @case / @default — scelta multipla.
  • @defer — caricamento lazy di blocchi (e componenti) al viewport o on interaction.

@if / @else

@if (showMessage()) {

Questo messaggio è visibile. Clicca il pulsante per nasconderlo.

} @else {

Messaggio nascosto. Clicca di nuovo per mostrarlo.

}
