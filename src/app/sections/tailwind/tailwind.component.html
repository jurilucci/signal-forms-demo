Tailwind CSS

Utility-first

Cos'è

Tailwind CSS è il framework utility-first supportato ufficialmente da Angular: nessun componente predefinito, ma puoi costruire card, badge, alert, form con le sole classi.

  • Setup: ng add @angular/tailwind o PostCSS + @use 'tailwindcss'.
  • Content detection per tree-shaking delle classi non usate.
