Angular MCP

Cos'è

L'Angular CLI MCP Server espone strumenti per integrare Angular con ambienti AI-assisted (es. Cursor, IDE con supporto MCP): ricerca documentazione, esempi, best practice, analisi progetti e generazione codice allineata ad Angular.

  • Documentazione e esempi ufficiali direttamente nel flusso dell'AI.
  • Validazione e suggerimenti coerenti con la versione del framework.
  • Utile per refactoring, migrazioni (es. OnPush/zoneless) e scaffolding.

Si configura come server MCP nel tuo editor; non è una dipendenza runtime dell'app.
